# Sem odevzdávejte úkol 2. - dark mode web


Vytvořte webovou stránku, která bude obsahovat nadpis, paragraf a líné, nic nedělající tlačítko. Všechny tyto prvky budou na středu. V levém horním rohu bude druhé tlačítko s ikonou měsíce, které bude přepínat web z light modu na dark mode. Pro web využijte své barevné schéma a fonty, které jste zvolili v prvním úkolu. Web uložte sem.
